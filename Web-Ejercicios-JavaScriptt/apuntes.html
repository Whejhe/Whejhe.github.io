Variables

Una variable es un “almacén con un nombre” para guardar datos

Para generar una variable en JavaScript, se usa la palabra clave let

let message;

Podemos combinar la declaración de la variable y su asignación en una sola línea

let message="Hello World!";

También podemos declarar variables múltiples en una sola línea:

let username="John Doe", password="secret", email="jdoe@me.com";

La versión de líneas múltiples es un poco más larga, pero se lee más fácil:


            let username="John Doe",
            let password="secret",
            let email="jdoe@me.com";

NOMBRAMIENTO DE VARIABLES

Existen dos limitaciones de nombre de variables en JavaScript:

  1. El nombre únicamente puede incluir letras, dígitos, o los símbolos $ y _.
  2. El primer carácter no puede ser un dígito
    3. 
                    let abc;        let $ = 1;      
                    let test123;    let _ = 2;

La Capitalización es Importante. No es lo mismo manzana que Manzana.

Letras que no son del alfabeto inglés están permitidas, pero no se recomiendan.

Es posible definir una variable sin var ni let.

VARIABLES - CONSTANTES

Se usa const en lugar de let.

No pueden ser alteradas

Se deben nombrar en Mayúsculas cuando se conocen su valor con antelación

const PI = 3.14

Ser una “constante” solo significa que el valor de la variable nunca cambia. Pero hay constantes que son conocidas previo a la ejecución (como el valor hexadecimal del color rojo) y hay constantes que son calculadas en el tiempo de ejecución, pero no cambian después de su asignación inicial.

TIPOS DE DATOS

Un valor en JavaScript siempre pertenece a un tipo de dato determinado

Hay ocho tipos de datos básicos en JavaScript.

Podemos almacenar un valor de cualquier tipo dentro de una variable. Por este motivo JavaScript se considera un lenguaje de tipado dinámico o débil

TIPOS DE DATOS - PRIMITIVOS

Number:
  • El tipo number representa tanto números enteros como de punto flotante.
  • Además de los números comunes, existen los llamados “valores numéricos especiales” que también pertenecen a este tipo de datos: Infinity, -Infinity y NaN.
  • Hacer matemáticas es “seguro” en JavaScript. El script nunca se detendrá con un error fatal (“morir”). En el peor de los casos, obtendremos NaN como resultado.

BigInt:
  • En JavaScript, el tipo “number” no puede representar valores enteros mayores que (2 53 -1) (eso es 9007199254740991), o menor que -(2 53 -1) para negativos
  • BigInt se agregó recientemente al lenguaje para representar enteros de longitud arbitraria
  • Un valor BigInt se crea agregando n al final de un entero:
    • 
                    //la "n" al final significa que es un BigInt
                    const bigInt = 1234567894390847029374121293n;

TIPOS DE DATOS - PRIMITIVOS

String:
  • Un string en JavaScript es una cadena de caracteres y debe colocarse entre comillas
  • En JavaScript, hay 3 tipos de comillas.
    1. Comillas dobles: "Hola".
    2. Comillas simples: 'Hola'.
    3. Backticks (comillas invertidas): `Hola`.
  • Las comillas dobles y simples son comillas “sencillas” (es decir, funcionan igual)
  • Los backticks son comillas de “funcionalidad extendida”. Nos permiten incrustar variables y expresiones en una cadena de caracteres encerrándolas en ${...}
    • 
                    //incrustar expresión
                    alert(`el resultado es ${1 + 2}`);
  • No existe el tipo carácter
