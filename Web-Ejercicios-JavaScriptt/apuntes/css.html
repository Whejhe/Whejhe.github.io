Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para controlar la presentación y el diseño de documentos HTML. Con CSS, puedes definir cómo se ven los elementos HTML en una página web.

En el mundo del desarrollo web, CSS es una herramienta esencial para crear sitios web atractivos y con estilo. Permite separar la estructura (HTML) del diseño y la apariencia (CSS) de una página, lo que facilita la administración y el mantenimiento del código.

En este tutorial, aprenderás los conceptos básicos de CSS, cómo aplicar estilos a elementos HTML y cómo crear reglas de estilo. Antes de comenzar, es importante que tengas un conocimiento básico de HTML, ya que CSS se utiliza para dar estilo a elementos HTML.

Selectores CSS

Los selectores CSS son patrones que se utilizan para seleccionar elementos HTML a los que se aplicarán estilos. Los selectores son fundamentales para determinar qué elementos se verán afectados por tus reglas de estilo. Aquí tienes ejemplos de diferentes tipos de selectores:

  • Selector de tipo: p selecciona todos los elementos `

    `. 

    
            p {
                color: blue;
            }
  • Selector de clase: .boton selecciona elementos con la clase "boton". 
    
            .boton {
                background-color: #ff9900;
            }
  • Selector de ID: #encabezado selecciona el elemento con el ID "encabezado". 
    
            #encabezado {
                font-size: 24px;
            }
  • Selector universal: * selecciona todos los elementos. 
    
            * {
                margin: 0;
                padding: 0;
            }
  • Selector de atributo: [type="text"] selecciona elementos con el atributo "type" igual a "text". 
    
            input[type="text"] {
                border: 1px solid #ccc;
            }
  • Selector de descendencia: ul li selecciona todos los elementos que están dentro de una lista desordenada 
      
            ul li {
                list-style: circle;
            }
    • Selector de grupo: Puedes agrupar varios selectores en una regla. 
      
            h1, h2, h3 {
                font-family: Arial, sans-serif;
            }

    Estos son algunos ejemplos de los tipos de selectores CSS disponibles. CSS ofrece una variedad de selectores avanzados que te permiten dirigirte a elementos de manera más específica, lo que es útil para aplicar estilos de manera precisa en tu sitio web.

  • Selector de Pseudo-clase `:active`: Selecciona elementos cuando están activos, como cuando un enlace es clicado. 
    
            a:active {
                color: purple;
            }
  • Selector de Pseudo-clase `:focus`: Selecciona elementos que tienen el foco, como un campo de entrada de texto que se selecciona con el cursor. 
    
            input:focus {
                border: 2px solid blue;
            }
  • Selector de Pseudo-clase `:visited`: Selecciona enlaces que ya han sido visitados por el usuario. 
    
            a:visited {
                color: gray;
            }
  • Selector de Pseudo-clase `:not()`: Permite seleccionar elementos que no cumplen con un cierto criterio. 
    
            p:not(.special) {
                font-size: 14px;
            }
  • Selector de Pseudo-elemento `::before` y `::after`: Permite agregar contenido antes o después de un elemento. 
    
            p::before {
                content: "Antes de: ";
            }
            
            p::after {
                content: "Después de.";
            }
  • Selector de Atributo de Subcadena `*`: Selecciona elementos cuyos atributos contienen una subcadena específica en cualquier parte del valor. 
    
            a[href*="example"] {
                color: orange;
            }
  • Selector de Atributo de Prefijo `^`: Selecciona elementos cuyos atributos comienzan con un valor específico. 
    
            input[type^="t"] {
                border: 1px solid green;
            }
  • Selector de Atributo de Sufijo `$`: Selecciona elementos cuyos atributos terminan con un valor específico. 
    
            a[href$=".pdf"] {
                font-weight: bold;
            }

Estos son algunos ejemplos de los selectores CSS más comunes y avanzados que puedes utilizar para aplicar estilos precisos a tus elementos HTML. La elección del selector adecuado depende de tus necesidades de diseño y estructura de la página web.

Propiedades CSS

Las propiedades CSS se utilizan para controlar el aspecto y el diseño de los elementos HTML. Aquí tienes una lista completa de propiedades organizadas por tipos:

Propiedades de Texto y Fuente:

  • font-family: Define la familia de fuentes para el texto.
  • font-size: Establece el tamaño de la fuente.
  • font-weight: Controla el grosor de la fuente (normal, negrita, etc.).
  • font-style: Especifica el estilo de la fuente (normal, cursiva, oblicua).
  • line-height: Define la altura de línea.
  • text-align: Alinea el texto (izquierda, derecha, centrado).
  • text-decoration: Agrega decoración de texto (subrayado, tachado, etc.).
  • text-transform: Cambia el capitalizado del texto (mayúsculas, minúsculas).
  • letter-spacing: Controla el espaciado entre caracteres.
  • word-spacing: Ajusta el espaciado entre palabras.

Propiedades de Color y Fondo:

  • color: Define el color del texto.
  • background-color: Establece el color de fondo.
  • opacity: Controla la opacidad del elemento.
  • background-image: Agrega una imagen de fondo.
  • background-size: Define el tamaño de la imagen de fondo.
  • background-repeat: Controla la repetición de la imagen de fondo.
  • background-position: Establece la posición de la imagen de fondo.

Propiedades de Borde:

  • border: Define el borde (ancho, estilo y color).
  • border-width: Establece el ancho del borde.
  • border-style: Especifica el estilo del borde (sólido, punteado, etc.).
  • border-color: Define el color del borde.
  • border-radius: Controla la curvatura de las esquinas del borde.

Propiedades de Espaciado y Dimensiones:

  • margin: Establece los márgenes del elemento.
  • padding: Define el espacio interno del elemento.
  • width: Establece el ancho del elemento.
  • height: Define la altura del elemento.
  • max-width: Establece el ancho máximo del elemento.
  • max-height: Define la altura máxima del elemento.
  • min-width: Establece el ancho mínimo del elemento.
  • min-height: Define la altura mínima del elemento.

Posicionamiento y Diseño:

  • position: Controla el posicionamiento de un elemento.
  • top: Establece la distancia desde la parte superior.
  • right: Establece la distancia desde la derecha.
  • bottom: Establece la distancia desde la parte inferior.
  • left: Establece la distancia desde la izquierda.
  • display: Controla cómo se muestra un elemento.
  • float: Establece el flotamiento de un elemento.
  • clear: Controla cómo los elementos deben comportarse en relación con los elementos flotantes.

Texto y Tipografía:

  • letter-spacing: Controla el espaciado entre caracteres.
  • word-spacing: Ajusta el espaciado entre palabras.
  • text-transform: Cambia el capitalizado del texto.
  • white-space: Define cómo se maneja el espacio en blanco dentro de un elemento.
  • line-height: Establece la altura de línea.
  • text-shadow: Agrega sombra al texto.

Transformaciones y Animaciones:

  • transform: Aplica transformaciones a un elemento, como rotación, escala y traslación.
  • animation: Define animaciones personalizadas.
  • transition: Controla transiciones suaves entre estados de un elemento.

Interacción y Comportamiento:

  • cursor: Cambia el cursor del mouse al pasar sobre un elemento.
  • pointer-events: Controla la interacción del puntero del mouse con un elemento.
  • user-select: Controla si se puede seleccionar texto en un elemento.

Listas y Marcadores:

  • list-style: Define el estilo de marcador de lista.
  • list-style-type: Establece el tipo de marcador de lista.

Fuentes Web (Web Fonts):

  • @font-face: Permite la incorporación de fuentes personalizadas en un sitio web.

Dimensionamiento Flexible (Flexbox y Grid):

  • display: Define cómo se muestra un elemento. En el caso de Flexbox, se usa para crear un contenedor flexible.
  • flex: Controla cómo se distribuye el espacio entre los elementos dentro de un contenedor flexible.
  • grid-template: Establece la estructura de filas y columnas en un contenedor de rejilla (Grid).
  • align-items: Alinea los elementos en el eje vertical en un contenedor flexible o de rejilla.
  • justify-content: Alinea los elementos en el eje horizontal en un contenedor flexible o de rejilla.

Fondos Avanzados:

  • background-image: Establece una imagen de fondo para un elemento.
  • background-size: Controla el tamaño de una imagen de fondo.
  • background-repeat: Define cómo se repite una imagen de fondo.
  • background-position: Establece la posición de una imagen de fondo.
  • background-attachment: Controla si una imagen de fondo se desplaza con el contenido o permanece fija.

Efectos Visuales:

  • box-shadow: Agrega sombra a un elemento.
  • text-overflow: Controla cómo se maneja el desbordamiento de texto en un elemento.
  • overflow: Controla cómo se muestra el contenido que desborda un elemento.
  • opacity: Establece la opacidad de un elemento.

Transformación de Box Model:

  • margin: Establece los márgenes externos de un elemento.
  • padding: Define los rellenos internos de un elemento.
  • border: Controla el tamaño y estilo de los bordes de un elemento.
  • box-sizing: Controla cómo se calcula el tamaño total de un elemento (incluyendo o excluyendo los márgenes y rellenos).
  • outline: Establece una línea de contorno alrededor de un elemento sin afectar al diseño.
