Variables en JavaScript

Una variable es un "almacén con un nombre" para guardar datos.

Para declarar una variable en JavaScript, se usa la palabra clave let:


        let mensaje;

Podemos declarar y asignar una variable en una sola línea:


        let mensaje = "Hola, mundo!";

También es posible declarar varias variables en una sola línea:


        let usuario = "Juan", password = "secreto", email = "juan@example.com";

La versión de líneas múltiples es más legible:


        let usuario = "Juan",
            password = "secreto",
            email = "juan@example.com";

Nombramiento de Variables

Existen dos limitaciones en los nombres de variables en JavaScript:

  1. El nombre puede incluir letras, dígitos y los símbolos $ y _.
  2. El primer carácter no puede ser un dígito.

        let abc; // Válido
        let $ = 1; // Válido
        let _ = 2; // Válido

La capitalización es importante; manzana y Manzana son diferentes.

Aunque se permiten letras que no son del alfabeto inglés, no se recomienda su uso.

También es posible definir una variable sin var ni let.

Tipos de Datos en JavaScript

En JavaScript, los valores pertenecen a tipos de datos específicos.

Existen ocho tipos de datos básicos en JavaScript:

  • Number: Representa números enteros y de punto flotante, incluyendo valores especiales como Infinity, -Infinity y NaN.
  • BigInt: Se utiliza para representar enteros de longitud arbitraria.
  • String: Representa cadenas de caracteres y puede definirse con comillas dobles, comillas simples o comillas invertidas (backticks).
  • Boolean: Tiene solo dos valores posibles: true y false.
  • null: Es un valor especial que no pertenece a ningún otro tipo de dato.
  • undefined: Indica que una variable no tiene un valor asignado.
  • Object: Se utiliza para estructuras de datos complejas.
  • Symbol: Se utiliza para crear identificadores únicos para los objetos.

El Operador typeof

El operador typeof se utiliza para determinar el tipo de un valor:

  • Como operador: typeof x.
  • Como una función: typeof(x).

Devuelve una cadena con el nombre del tipo, pero es importante notar que para null devuelve "object", lo cual es un error conocido en JavaScript.

Funciones en JavaScript

Una función en JavaScript es un bloque de código reutilizable que realiza una tarea específica.

Para definir una función, utilizamos la palabra clave function:


        function miFuncion() {
            // Código de la función
        }

Las funciones pueden recibir parámetros que actúan como variables dentro de la función:


        function saludar(nombre) {
            console.log(`Hola, ${nombre}!`);
        }
        saludar("Juan"); // Salida: "Hola, Juan!"

Las funciones pueden devolver un valor usando la palabra clave return:


        function suma(a, b) {
            return a + b;
        }
        const resultado = suma(2, 3); // resultado contendrá 5

Ámbito de las Variables

Las variables declaradas dentro de una función solo son visibles dentro de esa función, esto se llama ámbito local:


        function miFuncion() {
            let x = 10; // variable local
        }
        console.log(x); // Error: x no está definida aquí

Las variables declaradas fuera de cualquier función son globales y se pueden acceder desde cualquier parte del código:


        let y = 20; // variable global
        function otraFuncion() {
            console.log(y); // Se puede acceder a y desde aquí
        }

Funciones Flecha

Las funciones flecha son una forma más corta de definir funciones:


        const suma = (a, b) => a + b;
        const resultado = suma(2, 3); // resultado contendrá 5

Invocación de Funciones

Para llamar a una función, simplemente escribimos su nombre seguido de paréntesis:


        miFuncion(); // Llamada a la función miFuncion

Podemos pasar argumentos a la función dentro de los paréntesis:


        saludar("Ana"); // Llamada a la función saludar con el argumento "Ana"

Funciones como Expresiones

En JavaScript, las funciones también pueden ser tratadas como valores y asignadas a variables:


        const miFuncion = function() {
            // Código de la función
        };
        miFuncion(); // Llamada a la función a través de la variable

Operadores Básicos en JavaScript

Los operadores son símbolos que se utilizan para realizar operaciones en JavaScript.

Algunos de los operadores básicos incluyen:

  • Operadores Aritméticos: Se utilizan para realizar cálculos matemáticos.
    • 
        let suma = 5 + 3; // suma contiene 8
        let resta = 10 - 4; // resta contiene 6
        let multiplicacion = 6 * 2; // multiplicacion contiene 12
        let division = 8 / 4; // division contiene 2
  • Operadores de Asignación: Se utilizan para asignar valores a variables.
    • 
        let x = 10; // Asignación
        x += 5; // Incremento (x contiene 15)
        x -= 3; // Decremento (x contiene 12)
  • Operadores de Comparación: Se utilizan para comparar valores y devuelven un valor booleano (true o false).
    • 
        let a = 5;
        let b = 7;
        let igual = a === b; // false (igualdad)
        let noIgual = a !== b; // true (desigualdad)
        let mayorQue = a > b; // false (mayor que)
        let menorQue = a < b; // true (menor que)
        let mayorIgual = a >= b; // false (mayor o igual que)
        let menorIgual = a <= b; // true (menor o igual que)
  • Operadores Lógicos: Se utilizan para realizar operaciones lógicas y combinan valores booleanos.
    • 
        let p = true;
        let q = false;
        let and = p && q; // false (AND lógico)
        let or = p || q; // true (OR lógico)
        let not = !p; // false (NOT lógico)

Operadores de Comparación en JavaScript

Los operadores de comparación se utilizan para comparar valores en JavaScript.

Algunos ejemplos de operadores de comparación incluyen:

  • Operador de Igualdad (==): Compara si dos valores son iguales, sin tener en cuenta el tipo de dato.
  • Operador de Desigualdad (!=): Compara si dos valores no son iguales, sin tener en cuenta el tipo de dato.
  • Operador de Igualdad Estricta (===): Compara si dos valores son iguales y tienen el mismo tipo de dato.
  • Operador de Desigualdad Estricta (!==): Compara si dos valores no son iguales o tienen diferentes tipos de datos.
  • Operador Mayor que (>): Compara si un valor es mayor que otro.
  • Operador Menor que (<):< /strong> Compara si un valor es menor que otro.
  • Operador Mayor o Igual que (>=): Compara si un valor es mayor o igual que otro.
  • Operador Menor o Igual que (<=):< /strong> Compara si un valor es menor o igual que otro.

Estos operadores devuelven un valor booleano (true o false) según el resultado de la comparación.

Condicionales en JavaScript

Los condicionales son estructuras de control que permiten tomar decisiones en un programa JavaScript.

JavaScript admite varios tipos de condicionales:

  • if: Se utiliza para ejecutar un bloque de código si una condición es verdadera. 
    • 
        if (condicion) {
            // Código a ejecutar si la condición es verdadera
        }
  • else: Se utiliza junto con un bloque if para ejecutar un bloque de código si la condición es falsa.
    • 
        if (condicion) {
            // Código a ejecutar si la condición es verdadera
        } else {
            // Código a ejecutar si la condición es falsa
        }
  • else if: Permite evaluar múltiples condiciones en secuencia.
    • 
        if (condicion1) {
            // Código a ejecutar si la condición1 es verdadera
        } else if (condicion2) {
            // Código a ejecutar si la condición2 es verdadera
        } else {
            // Código a ejecutar si ninguna condición es verdadera
        }
  • switch: Se utiliza para evaluar una expresión y ejecutar diferentes bloques de código según el valor de la expresión.
    • 
        switch (expresion) {
            case valor1:
                // Código a ejecutar si expresion === valor1
                break;
            case valor2:
                // Código a ejecutar si expresion === valor2
                break;
            default:
                // Código a ejecutar si ninguna coincidencia
        }

Operadores de Comparación en Condicionales

En los condicionales, se utilizan operadores de comparación para evaluar condiciones:

  • ==: Compara si dos valores son iguales, convirtiendo tipos si es necesario.
  • ===: Compara si dos valores son iguales y tienen el mismo tipo.
  • !=: Compara si dos valores no son iguales, convirtiendo tipos si es necesario.
  • !==: Compara si dos valores no son iguales o tienen tipos diferentes.
  • >: Compara si un valor es mayor que otro.
  • <: Compara si un valor es menor que otro.
  • >=: Compara si un valor es mayor o igual que otro.
  • <=: Compara si un valor es menor o igual que otro.

Condicionales Avanzados en JavaScript

Además de los condicionales básicos, JavaScript ofrece características avanzadas para controlar el flujo de un programa de manera más sofisticada:

  • Operador Ternario: El operador ternario (también conocido como operador condicional) permite realizar una evaluación condicional en una sola línea.
    • 
        const resultado = (condicion) ? "Verdadero" : "Falso";
  • Condiciones Anidadas: Puedes anidar múltiples estructuras condicionales dentro de otras para manejar situaciones más complejas.
    • 
        if (condicion1) {
            if (condicion2) {
                // Código si ambas condiciones son verdaderas
            } else {
                // Código si la primera condición es verdadera pero la segunda es falsa
            }
        } else {
            // Código si la primera condición es falsa
        }
  • Operadores Lógicos: JavaScript proporciona operadores lógicos como `&&` (AND), `||` (OR) y `!` (NOT) para combinar y negar condiciones.
    • 
        if (condicion1 && condicion2) {
            // Código si ambas condiciones son verdaderas
        }
        
        if (condicion1 || condicion2) {
            // Código si al menos una condición es verdadera
        }
        
        if (!condicion) {
            // Código si la condición es falsa
        }
  • Switch con Múltiples Casos: Puedes agrupar casos en un bloque `switch` para ejecutar el mismo código para múltiples valores.
    • 
        switch (valor) {
            case 1:
            case 2:
                // Código a ejecutar si valor es 1 o 2
                break;
            case 3:
                // Código a ejecutar si valor es 3
                break;
            default:
                // Código a ejecutar si no coincide ningún caso
        }

Condicionales Difíciles de Entender

A medida que los programas crecen, los condicionales complejos pueden volverse difíciles de entender y mantener. Es importante escribir código legible y modular para facilitar el mantenimiento:

  • Refactorización: Divide condicionales largos en funciones más pequeñas y nombradas para mejorar la legibilidad.
  • Comentarios: Agrega comentarios descriptivos para explicar la lógica detrás de las condiciones complejas.
  • Tablas de Decisión: En lugar de condicionales anidadas, considera el uso de tablas de decisión para mapear entradas a resultados.

Bucles en JavaScript

Los bucles son estructuras de control que permiten ejecutar un bloque de código repetidamente. En JavaScript, existen tres tipos principales de bucles:

  • 1. Bucle For: El bucle for se utiliza para ejecutar un bloque de código un número específico de veces.
    • 
        for (let i = 0; i < 5; i++) {
            // Código a repetir
        }
  • 2. Bucle While: El bucle while se utiliza cuando no sabes cuántas veces se repetirá el código, pero la repetición depende de una condición.
    • 
        let contador = 0;
        while (contador < 5) {
            // Código a repetir
            contador++;
        }
  • 3. Bucle Do-While: El bucle do-while es similar al while, pero garantiza que el código se ejecute al menos una vez antes de verificar la condición.
    • 
        let numero = 0;
        do {
            // Código a repetir al menos una vez
            numero++;
        } while (numero < 5);

Control de Bucles

En bucles, es importante tener control sobre cuándo y cómo se detiene la repetición. Aquí hay algunas declaraciones clave:

  • 1. Break: La declaración `break` se utiliza para salir inmediatamente de un bucle. 
    • 
        for (let i = 0; i < 10; i++) {
            if (i === 5) {
                break; // Sale del bucle cuando i es igual a 5
            }
        }
  • 2. Continue: La declaración `continue` se utiliza para saltar una iteración del bucle y continuar con la siguiente.
    • 
        for (let i = 0; i < 5; i++) {
            if (i === 2) {
                continue; // Salta la iteración cuando i es igual a 2
            }
            // Código que se ejecuta en todas las iteraciones excepto cuando i es igual a 2
        }

Bucles Avanzados

Además de los bucles básicos, JavaScript ofrece funcionalidades avanzadas para tratar con estructuras de datos y tareas más complejas:

  • 1. Bucles For-In: Los bucles for-in se utilizan para iterar sobre propiedades de objetos.
    • 
        const persona = {
            nombre: 'Juan',
            edad: 30,
            ocupacion: 'Desarrollador'
        };
        
        for (let propiedad in persona) {
            console.log(propiedad + ': ' + persona[propiedad]);
        }
  • 2. Bucles For-Of: Los bucles for-of se utilizan para iterar sobre elementos de estructuras de datos iterables como arrays.
    • 
        const colores = ['Rojo', 'Verde', 'Azul'];
        
        for (let color of colores) {
            console.log(color);
        }
  • 3. Bucles Recursivos: Los bucles recursivos son funciones que se llaman a sí mismas para repetir una tarea de manera iterativa.
    • 
        function contarHasta(numero) {
            if (numero <= 0) {
                return;
            }
            console.log(numero);
            contarHasta(numero - 1);
        }
        
        contarHasta(5);

Mejoras en ES6+

Con la introducción de ECMAScript 6 (ES6) y versiones posteriores, JavaScript ha introducido características adicionales para simplificar y mejorar el uso de bucles, como el bucle `for...of`, `forEach`, `map`, `filter`, y `reduce` en arrays.

Objetos en JavaScript

Los objetos son una parte fundamental de JavaScript y se utilizan para almacenar y organizar datos de manera estructurada. Un objeto es una colección de propiedades, donde cada propiedad tiene un nombre (clave) y un valor asociado.


        const persona = {
            nombre: 'Juan',
            edad: 30,
            ocupacion: 'Desarrollador'
        };

En el ejemplo anterior, `persona` es un objeto con tres propiedades: `nombre`, `edad` y `ocupación`. Cada propiedad tiene un valor correspondiente.

Acceder a Propiedades de Objetos

Puedes acceder a las propiedades de un objeto utilizando la notación de punto o la notación de corchetes: 


        const nombre = persona.nombre; // Notación de punto
        const edad = persona['edad'];  // Notación de corchetes

Ambos enfoques son equivalentes y te permiten obtener el valor de una propiedad específica.

Añadir y Modificar Propiedades

Puedes añadir nuevas propiedades a un objeto o modificar las existentes en cualquier momento:


        persona.correo = 'juan@example.com'; // Añadir nueva propiedad
        persona.edad = 31;                  // Modificar propiedad existente

Eliminar Propiedades

Para eliminar una propiedad de un objeto, puedes utilizar el operador `delete`:


        delete persona.ocupacion; // Eliminar la propiedad 'ocupacion'

Iterar a través de Propiedades

Puedes iterar a través de las propiedades de un objeto utilizando bucles `for...in`:


        for (let propiedad in persona) {
            console.log(propiedad + ': ' + persona[propiedad]);
        }

Esto te permite acceder a todas las propiedades del objeto y sus valores.

Objetos Anidados

Los objetos pueden contener otros objetos como propiedades. Esto se conoce como objetos anidados:


        const libro = {
            titulo: 'JavaScript Avanzado',
            autor: {
                nombre: 'Ana',
                apellido: 'Gómez'
            }
        };

En este ejemplo, el objeto `libro` tiene una propiedad `autor` que es otro objeto con sus propias propiedades.

Métodos del Objeto Number en JavaScript

El objeto `Number` en JavaScript proporciona una variedad de métodos para realizar operaciones y manipulaciones con números. Estos métodos facilitan tareas comunes relacionadas con números.



1. Método toFixed()

El método `toFixed()` se utiliza para formatear un número con un número específico de decimales y devuelve una cadena de caracteres.


        const numero = 5.12345;
        const numeroFormateado = numero.toFixed(2); // Devuelve "5.12"

2. Método parseInt() y parseFloat()

Los métodos `parseInt()` y `parseFloat()` se utilizan para analizar una cadena y convertirla en un número entero o de punto flotante, respectivamente.


        const cadenaEntera = '42';
        const cadenaDecimal = '3.14';
        
        const numeroEntero = parseInt(cadenaEntera);     // Devuelve 42
        const numeroDecimal = parseFloat(cadenaDecimal); // Devuelve 3.14

3. Método isNaN()

El método `isNaN()` verifica si un valor es NaN (No es un número).


        const valor = 10;
        const esNaN = isNaN(valor); // Devuelve false

4. Método isFinite()

El método `isFinite()` verifica si un valor es un número finito.


        const valor = 10;
        const esFinito = isFinite(valor); // Devuelve true

5. Método toPrecision()

El método `toPrecision()` formatea un número con una precisión específica y devuelve una cadena de caracteres.


        const numero = 123.45678;
        const numeroFormateado = numero.toPrecision(4); // Devuelve "123.5"

6. Método toLocaleString()

El método `toLocaleString()` convierte un número en una representación de cadena de caracteres formateada según la configuración regional actual del usuario.


        const numero = 1234567.89;
        const numeroFormateado = numero.toLocaleString(); // Dependiendo de la configuración regional, podría devolver "1,234,567.89"

7. Método toString()

El método `toString()` convierte un número en una cadena de caracteres.


        const numero = 42;
        const cadena = numero.toString(); // Devuelve "42"

8. Método valueOf()

El método `valueOf()` devuelve el valor numérico de un objeto `Number`.


        const numero = new Number(42);
        const valorNumerico = numero.valueOf(); // Devuelve 42

9. Propiedad Number.MAX_VALUE y Number.MIN_VALUE

Estas propiedades del objeto `Number` representan el valor máximo y mínimo que se pueden representar en JavaScript.


        const maximoValor = Number.MAX_VALUE; // Valor máximo en JavaScript
        const minimoValor = Number.MIN_VALUE; // Valor mínimo en JavaScript

10. Propiedad Number.POSITIVE_INFINITY y Number.NEGATIVE_INFINITY

Estas propiedades del objeto `Number` representan los valores infinito positivo e infinito negativo.


        const infinitoPositivo = Number.POSITIVE_INFINITY; // Infinito positivo
        const infinitoNegativo = Number.NEGATIVE_INFINITY; // Infinito negativo

11. Propiedad Number.NaN

La propiedad `Number.NaN` representa el valor NaN (No es un número).


        const noEsUnNumero = Number.NaN; // NaN

Métodos del Objeto String en JavaScript

El objeto `String` en JavaScript proporciona una variedad de métodos para trabajar con cadenas de caracteres. Estos métodos permiten realizar manipulaciones, búsquedas y transformaciones en las cadenas.

1. Método length

La propiedad `length` devuelve la longitud de una cadena, es decir, la cantidad de caracteres que contiene.


        const cadena = 'Hola, mundo!';
        const longitud = cadena.length; // Devuelve 12

2. Método toUpperCase() y toLowerCase()

Los métodos `toUpperCase()` y `toLowerCase()` se utilizan para convertir una cadena a letras mayúsculas o minúsculas, respectivamente.


        const cadena = 'Hola, Mundo!';
        const mayusculas = cadena.toUpperCase(); // Devuelve "HOLA, MUNDO!"
        const minusculas = cadena.toLowerCase(); // Devuelve "hola, mundo!"

3. Método trim()

El método `trim()` elimina los espacios en blanco al principio y al final de una cadena.


        const cadena = '  Hola, mundo!  ';
        const cadenaLimpia = cadena.trim(); // Devuelve "Hola, mundo!"

4. Método concat()

El método `concat()` se utiliza para concatenar dos o más cadenas y crear una nueva cadena.


        const cadena1 = 'Hola, ';
        const cadena2 = 'mundo!';
        const resultado = cadena1.concat(cadena2); // Devuelve "Hola, mundo!"

5. Método indexOf() y lastIndexOf()

Los métodos `indexOf()` y `lastIndexOf()` se utilizan para buscar la posición de una subcadena dentro de una cadena. `indexOf()` devuelve la primera coincidencia y `lastIndexOf()` la última.


        const cadena = 'Hola, mundo!';
        const posicion1 = cadena.indexOf('mundo'); // Devuelve 6
        const posicion2 = cadena.lastIndexOf('o');   // Devuelve 10

6. Método slice()

El método `slice()` extrae una porción de una cadena y devuelve una nueva cadena con los caracteres seleccionados.


        const cadena = 'JavaScript';
        const subcadena = cadena.slice(0, 4); // Devuelve "Java"

7. Método replace()

El método `replace()` busca una subcadena y la reemplaza por otra en una cadena. Solo reemplaza la primera coincidencia a menos que se utilice una expresión regular con la bandera global (`/g`).


        const cadena = 'Hola, mundo!';
        const nuevaCadena = cadena.replace('mundo', 'amigos'); // Devuelve "Hola, amigos!"

8. Método split()

El método `split()` divide una cadena en un array de subcadenas basado en un delimitador especificado.


        const cadena = 'Manzana,Plátano,Uva';
        const frutas = cadena.split(','); // Devuelve ["Manzana", "Plátano", "Uva"]

9. Método substring() y substr()

Los métodos `substring()` y `substr()` extraen una porción de una cadena. `substring()` se basa en índices y `substr()` en posición inicial y longitud.


        const cadena = 'JavaScript';
        const subcadena1 = cadena.substring(0, 4); // Devuelve "Java"
        const subcadena2 = cadena.substr(4, 4);    // Devuelve "Scri"

10. Método includes(), startsWith() y endsWith()

Estos métodos se utilizan para verificar si una cadena contiene otra subcadena. `includes()` verifica si la cadena contiene la subcadena en cualquier posición, mientras que `startsWith()` y `endsWith()` verifican si la cadena comienza o termina con la subcadena, respectivamente.


        const cadena = 'Hola, mundo!';
        const incluyeMundo = cadena.includes('mundo');         // Devuelve true
        const empiezaConHola = cadena.startsWith('Hola');       // Devuelve true
        const terminaConExclamacion = cadena.endsWith('!');     // Devuelve true

11. Método charAt() y charCodeAt()

Los métodos `charAt()` y `charCodeAt()` se utilizan para acceder a caracteres individuales en una cadena. `charAt()` devuelve el carácter en una posición dada, y `charCodeAt()` devuelve el valor Unicode del carácter en esa posición.


        const cadena = 'JavaScript';
        const caracter = cadena.charAt(0);         // Devuelve "J"
        const codigoUnicode = cadena.charCodeAt(0); // Devuelve 74 (valor Unicode de "J")

12. Método repeat()

El método `repeat()` crea y devuelve una nueva cadena que contiene la cadena original repetida un número especificado de veces.


        const cadena = 'Hola';
        const repetida = cadena.repeat(3); // Devuelve "HolaHolaHola"

Expresiones Regulares en JavaScript

Las expresiones regulares, también conocidas como regex o regexp, son patrones que se utilizan para buscar y manipular texto basado en ciertas reglas. En JavaScript, puedes utilizar el objeto `RegExp` o escribir patrones directamente utilizando literales de expresiones regulares.

Creación de Expresiones Regulares

Hay dos formas de crear expresiones regulares en JavaScript:


        // Utilizando el objeto RegExp
        const regex1 = new RegExp('patrón');
        // Utilizando el literal de expresión regular
        const regex2 = /patrón/;

El segundo método es más común y se usa para patrones fijos.

Metacaracteres

Los metacaracteres son símbolos especiales que tienen un significado especial en una expresión regular. Aquí hay algunos de los metacaracteres más comunes:

  • .: Coincide con cualquier carácter excepto una nueva línea.
  • ^: Coincide con el inicio de una línea.
  • $: Coincide con el final de una línea.
  • *: Coincide con 0 o más repeticiones del elemento anterior.
  • +: Coincide con 1 o más repeticiones del elemento anterior.
  • ?: Coincide con 0 o 1 repetición del elemento anterior.
  • {n}: Coincide exactamente con n repeticiones del elemento anterior.
  • {n,}: Coincide con al menos n repeticiones del elemento anterior.
  • {n,m}: Coincide con entre n y m repeticiones del elemento anterior.
  • [abc]: Coincide con uno de los caracteres dentro de los corchetes.
  • [^abc]: Coincide con cualquier carácter que no esté dentro de los corchetes.

Expresiones Regulares con Literales

Los literales de expresiones regulares se crean encerrando el patrón entre barras diagonales. Por ejemplo:


        const regex = /patrón/;

Puedes usar los metacaracteres y cuantificadores dentro de los literales de expresiones regulares.

Métodos de Expresiones Regulares

En JavaScript, puedes utilizar métodos como `test()` y `exec()` con expresiones regulares para buscar coincidencias en cadenas de texto.


        const texto = 'Ejemplo de texto con una palabra clave.';
        const regex = /palabra/;
        const resultado = regex.test(texto); // Devuelve true si encuentra una coincidencia

Expresiones Regulares con Flags

Puedes agregar banderas a una expresión regular para controlar su comportamiento. Algunas banderas comunes incluyen:

  • i: Hace que la expresión regular sea insensible a mayúsculas y minúsculas.
  • g: Busca todas las coincidencias en lugar de detenerse en la primera.
  • m: Permite que ^ y $ coincidan con el inicio y el final de cada línea, en lugar de solo el inicio y el final del texto completo.

        const texto = 'Ejemplo de texto con una Palabra clave.';
        const regex = /palabra/i; // La 'i' hace que sea insensible a mayúsculas/minúsculas
        const resultado = regex.test(texto); // Devuelve true

Grupos y Capturas

Puedes usar paréntesis para crear grupos en una expresión regular. Esto permite realizar capturas y operaciones más avanzadas. Por ejemplo:


        const texto = 'Fecha de nacimiento: 01-02-1990';
        const regex = /(\d{2})-(\d{2})-(\d{4})/;
        const resultado = regex.exec(texto); // Devuelve un array con las coincidencias

Expresiones Regulares Predefinidas

JavaScript también proporciona expresiones regulares predefinidas para buscar patrones comunes, como direcciones de correo electrónico, números de teléfono, etc. Puedes encontrar estas expresiones regulares en la documentación de JavaScript.

Expresiones Regulares son Poderosas

Las expresiones regulares son una herramienta poderosa para trabajar con texto en JavaScript. Pueden ser útiles en la validación de formularios, búsqueda y manipulación de cadenas, análisis de texto y mucho más.

Arrays en JavaScript

Un array es una estructura de datos que permite almacenar múltiples valores en una sola variable. En JavaScript, los arrays pueden contener elementos de diferentes tipos de datos.

Creación de Arrays

Para crear un array en JavaScript, puedes utilizar la siguiente sintaxis:


        const miArray = []; // Array vacío
        const numeros = [1, 2, 3, 4, 5]; // Array con elementos

Acceso a Elementos

Los elementos de un array se acceden mediante un índice, comenzando desde 0 para el primer elemento:


        const numeros = [1, 2, 3, 4, 5];
        const primerElemento = numeros[0]; // Accede al primer elemento (1)

Propiedad length

La propiedad length te permite conocer la cantidad de elementos en un array:


        const numeros = [1, 2, 3, 4, 5];
        const cantidadElementos = numeros.length; // Devuelve 5

Operaciones con Arrays

JavaScript proporciona numerosos métodos para operar y modificar arrays:

  • push(): Agrega un elemento al final del array.
  • pop(): Elimina el último elemento del array.
  • unshift(): Agrega un elemento al principio del array.
  • shift(): Elimina el primer elemento del array.
  • splice(): Permite realizar operaciones avanzadas de inserción, eliminación y reemplazo de elementos.
  • concat(): Combina dos o más arrays para crear uno nuevo.
  • slice(): Crea una copia superficial (shallow copy) de un subconjunto de un array.

Recorriendo un Array

Puedes recorrer los elementos de un array utilizando bucles como for o for...of:


        const numeros = [1, 2, 3, 4, 5];
        for (let i = 0; i < numeros.length; i++) {
            console.log(numeros[i]);
        }
        // O usando for...of
        for (const numero of numeros) {
            console.log(numero);
        }

Arrays Multidimensionales

Un array multidimensional es un array que contiene otros arrays. Puedes crear matrices (matrices bidimensionales) para representar datos tabulares:


        const matriz = [
            [1, 2, 3],
            [4, 5, 6],
            [7, 8, 9]
        ];
        const elemento = matriz[1][2]; // Accede al valor 6

Métodos Funcionales

JavaScript también proporciona métodos funcionales como map(), filter() y reduce() para realizar operaciones más complejas en arrays y transformar los datos.

Arrays en JavaScript son Dinámicos

Los arrays en JavaScript son dinámicos, lo que significa que puedes cambiar su tamaño y contenido en tiempo de ejecución:


        const frutas = ['manzana', 'plátano', 'uva'];
        frutas.push('naranja'); // Agrega 'naranja' al final
        frutas.pop(); // Elimina el último elemento ('naranja')
